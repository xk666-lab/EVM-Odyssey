## 💡 练习建议

### 1. 先独立思考

不要马上看答案：
1. 读题目
2. 思考5-10分钟
3. 尝试写代码
4. 卡住了再看提示
5. 实在不会再看答案

### 2. 对比答案

看完答案后：
- 对比自己的实现
- 找出差异点
- 理解为什么答案更好
- 重写一遍

### 3. 举一反三

每个练习后：
- 尝试改进答案
- 添加新功能
- 优化Gas
- 增强安全性
